# 🎬 Movie Recommendation System

This project is a multi-engine movie recommendation system that combines different machine learning approaches to provide personalized and hybrid recommendations.

---

## 🚀 Project Overview

The system integrates multiple recommendation strategies:

- Content-Based Filtering
- Metadata-Based Recommendation
- Collaborative Filtering
- Sequence-Based (LSTM)
- Visual Recommendation
- Popularity-Based Ranking
- Hybrid Model

Each module is developed by a specialized engineer to ensure modularity and scalability.

---

## 🧠 System Architecture

🔧 Data Engineer - Data collection and preprocessing  
🤖 ML Engineer (Content) - Content & metadata processing  
👥 ML Engineer (Collaborative) - Collaborative filtering & LSTM  
🖼️ Computer Vision Engineer - Visual similarity models  
⚙️ Backend Engineer - Hybrid model integration  
🎨 Frontend Engineer - Streamlit web application

## ⚙️ Features

- Data extraction from TMDB dataset
- Advanced preprocessing pipeline
- Multiple recommendation algorithms
- Hybrid recommendation engine
- Evaluation metrics for model performance
- Interactive web interface using Streamlit

---

## 📊 How It Works

1. Data is collected and cleaned using TMDB scraper
2. Multiple ML models are trained independently
3. Results are combined into a hybrid recommender
4. User interacts through a Streamlit UI
5. System generates personalized movie recommendations
